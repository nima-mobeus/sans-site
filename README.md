# Hosted Site Template

This is a template for hosted websites on the Mobeus Voice AI Platform. It creates a Next.js site with Tailwind CSS that comes pre-configured for voice AI integration.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Structure

- `src/app/` — Next.js App Router pages
- `src/components/` — Reusable React components
- `public/` — Static assets

## Deployment

When you push to the `main` branch, your site will be automatically rebuilt and deployed.
